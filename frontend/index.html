Snapshot + AOF

Snapshot 파일 확인 중 마지막 저장 시각을 불러오는 중입니다. 파일 상태를 불러오는 중입니다.
AOF 파일 확인 중 마지막 기록 시각을 불러오는 중입니다. 파일 상태를 불러오는 중입니다.
복구 흐름 Snapshot -> AOF Replay 시작 시 Snapshot을 먼저 불러오고, 그 뒤 AOF 변경분을 replay 해 최종 상태를 복원합니다. 종료 시 Snapshot 저장 · 비정상 종료 시 AOF가 마지막 변경분을 살립니다.
